h1. American Express Open Forum Gem

"Open Forum":http://developer.openforum.com/ is a site that offers content about Small Business.
It is associated with American Express's OPEN CARD - a credit card used by Small Business Owners.

h2. The Gem

This is gem uses "httparty":https://github.com/jnunemaker/httparty and "crack":https://github.com/jnunemaker/crack to wrap calls to open forum.
It then uses "hashery's":http://rubyworks.github.com/hashery "OpenCascade":http://www.ruby-doc.org/gems/docs/h/hashery-1.5.0/OpenCascade.html class to turn the API's hash response into callable methods.
